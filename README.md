# 欢迎使用IteBrowser插件。

## 调用方法

您只要使用以下HTML自带方法，就可以绑定在IE8或更老的浏览器显示IteBrowser插件。

```html
  <!--[if Ite IE 8]><br>
    <iframe src="itebrowser/itebrowser.min.html"></iframe><br>
  <!--[endif]>
```

您可以在代码中找到 `IE 8` 字段，然后修改您要显示IteBrowser插件的浏览器。
